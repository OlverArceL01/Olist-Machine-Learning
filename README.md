# Olist Machine Learning App
Ir a [`docs/index.md`](https://github.com/OlverArceL01/olist-ml/blob/main/docs/index.md) para ver la documentación del proyecto.
## Setup
1. Crear un entorno con [Poetry](https://python-poetry.org/).
2. Instalar las dependencias del proyecto:
   ```bash
   poetry install
   ```
3. Activar el entorno
   ```bash
  poetry shell
   ```
4. Se require tener instalado [Google Cloud CLI](https://docs.cloud.google.com/sdk/docs/install-sdk)
5. Abrir el archivo `conf/config.yaml` para ajustar nombres y rutas de los recursos de Google Cloud, y rutas de los archivos a utilizar.
